# Job Application Tracker
The Job Application Tracker is a professional-grade full-stack web application designed to help job seekers organize their career search. Built with a modern tech stack focusing on performance and scalability, it features a complex, interactive dashboard for managing application states.


## 🚀 Key Features

* **Interactive Kanban Board:** A complex drag-and-drop interface for tracking job stages (Applied, Interviewing, Offered, etc.).
* **Secure Authentication:** User accounts and session management powered by Better Auth.
* **Real-time Persistence:** Application states and board positions are saved instantly to a MongoDB cluster.
* **Server-Side Excellence:** Optimized using Next.js Server Components for faster page loads and improved SEO.
* **Responsive Design:** A sleek, dark-themed UI built with Tailwind CSS and Shadcn UI components.

## 🛠️ Tech Stack

* **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/)
* **Database:** [MongoDB](https://www.mongodb.com/) (via Mongoose)
* **Auth:** [Better Auth](https://www.better-auth.com/)
* **Deployment:** [Vercel](https://vercel.com/)

## 📂 Project Structure

```text
├── app/               # Next.js App Router (Pages, Layouts, APIs)
├── components/        # Shared React components (UI/UX)
│   └── ui/            # Shadcn UI primitives
├── lib/               # Utility functions and Database configurations
├── models/            # MongoDB/Mongoose schemas
└── public/            # Static assets

```
